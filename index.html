wired-php by efphe

Php Client for WuBook Booking Online and Channel Manager API

WuBook (http://en.wubook.net/, http://wubook.net/) povides Xmlrpc Api to implement a custom booking online and/or Channel Manager service. This library is an Xmlrpc client developed to communicate with WuBook web services.

Dependencies

Php >= 5.0 http://phpxmlrpc.sourceforge.net/

Install

Library is a small library: put the unique file on appropriate place and use it.
